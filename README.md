# OS Theory Assignment 1: System Call Operations

This repository contains the source code for Assignment 1 of the Operating Systems Theory course (4th Semester).

## project Description
The assignment demonstrates the use of system calls in C to interact with hardware at a low level. It includes operations for file handling, process management, and other system-level interactions.

## Key Files
- `24P-0706_Aazan_Noor_khuwaja_BSCS_4B.c`: Main source code file implementing the system call operations.
- `assgt1_helper.c`: Helper functions for the assignment.

## Compilation and Execution
To compile the code, use the following command:
```bash
gcc 24P-0706_Aazan_Noor_khuwaja_BSCS_4B.c -o assignment1
```

To run the executable:
```bash
./assignment1
```
